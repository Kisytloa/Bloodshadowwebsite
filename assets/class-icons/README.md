# Icônes de classes Lost Ark

Placez ici vos PNG d'icônes officielles (format 320×320 recommandé).

Pour activer les icônes locales, ajoutez avant le chargement de la page :
```html
<script>window.USE_LOCAL_ICONS = true;</script>
```

**Noms de fichiers** (minuscules, tirets) :
- `berserker.png`, `paladin.png`, `destructeur.png`, `gunlancer.png`
- `slayer.png`, `valkyrie.png` (utilise paladin.png si absent)
- `sorciere.png`, `arcaniste.png`, `barde.png`, `invocatrice.png`
- `elementiste.png`, `spirite.png`, `lanciere.png`, `pugiliste.png`
- `striker.png`, `breaker.png`
- `sanguelame.png`, `demoniste.png`, `faucheuse.png`, `souleater.png`
- `sagittaire.png`, `artilleur.png`, `machiniste.png`, `fusiliere.png`
- `artiste.png`, `heletempete.png`, `ame-sauvage.png`
- `guardian-knight.png`, `dragon-knight.png`

Si un fichier local existe, il sera utilisé à la place du CDN Fandom.
